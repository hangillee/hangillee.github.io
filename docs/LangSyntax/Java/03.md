## 숫자
Java에는 정수와 실수, 두 가지 형태의 숫자 자료형이 존재한다. 우리가 흔히 알고 있는 **숫자** 그 자체를 저장할 수 있는 자료형이다.

| 정수형                                           |
| 이름   | 크기  | 표현 범위                        |
|:-------|:------|:--------------------------------|
| int    | 32bit | -2,147,483,638 ~ +2,147,483,647 |
