# HTTP
## HyperText Transfer Protocol

HTTP는 WWW(World Wide Web)상에서 데이터를 주고 받을 수 있는 프로토콜입니다.

HTTP 카테고리의 문서들은 인프런(https://inflearn.com)에서 학습할 수 있는 **"우아한형제들 최연소 기술이사 김영한의 스프링 완전 정복"** 로드맵을 수강한 후, 개인적으로 내용을 정리하는 문서들입니다.
