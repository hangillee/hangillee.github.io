# Spring
## 스프링 프레임워크

Java 기반의 오픈 소스 웹 프레임워크입니다. 현재는 주로 Spring Boot를 통해 Spring을 사용합니다.

Spring 카테고리의 문서들은 인프런(https://inflearn.com)에서 학습할 수 있는 **"우아한형제들 최연소 기술이사 김영한의 스프링 완전 정복"** 로드맵을 수강한 후, 개인적으로 내용을 정리하는 문서들입니다.
